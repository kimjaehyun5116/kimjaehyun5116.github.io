## 🚀 내 풀이 ⭕

```java
public class Main {
    public static void main(String[] args){
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```

이번 문제는 딱히 포스팅 이랄 것이 없다. 아주 간단한 출력 문제이다보니 모양을 따라서 그리는것만 잘하면 쉽게 맞출 수 있다<br/>
